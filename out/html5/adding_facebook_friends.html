Adding Facebook Friends

This document describes how to add facebook friends.

You should have your facebook profile or create one before you begin this task .

1. You can add new facebook friends using the steps mentioned in this task.

2. If you want to add new friends on facebook, use this task.

3. New facebook friends can be added to your facebook account using these steps.

  1. Tap the friends button to see a list of people you may know.
  2. If you see someone you know, you can send them a friend request by tapping Add.
  3. Tap this button on top to see more options.
  4. From here you can see Friend suggestions, Search for friends, Connect with friends through your contacts, and View your current friends.
  5. You can respond to Friend requests here too.
  6. To accept a friend request, tap Confirm.
  7. You can tap for friends by tapping Search, then typing in a friend's name.
